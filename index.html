Bienvenido a Arbópolis

El principal objetivo de Arbópolis es desarrollar una aplicación móvil que fomente la sostenibilidad ambiental, permitiendo a los usuarios identificar, proteger y plantar árboles en Bogotá.

Objetivos específicos

  • Desarrollar una base de datos de árboles protegidos.
  • Crear un sistema de recompensas para la participación ciudadana.
  • Organizar campañas de sensibilización para la comunidad.
  • Proveer soporte técnico y actualizaciones continuas para la aplicación.
